## SQL Concepts Review

1. WHERE filters rows based on a condition.
2. LIKE '%Software%' searches for text that contains Software anywhere in the selected column.
3. ORDER BY sorts the result set by a selected column. ASC means ascending order.
4. COUNT(*) counts every row returned by the query, including rows that contain NULL values.
5. LIMIT 3 returns only the first 3 rows from the result set. It is most useful when combined with ORDER BY.
6. NULL means the value is missing, unknown, or not yet provided.
7. SELECT specific columns is better than SELECT * when we only need certain data.
